  중요한 문법 같은 경우에는 수업 시간에 당연하게도 설명을 드리겠지만,
  다들 언어 하나씩은 하실 수 있고, 초반 러닝 커브는 높지 않기 때문에 너무 기본적인 문법은 건너뛰고 진행할 예정입니다.
